What Are RustPress Plugins?

RustPress plugins are modular extensions that add new functionality to your CMS. Built with Rust or WebAssembly, they're secure, fast, and easy to install.

  • Rust Native - Plugins written in Rust for maximum performance
  • WASM Support - Run WebAssembly plugins in a sandboxed environment
  • Hot Reload - Develop plugins without restarting the server
  • Hook System - Extend any part of RustPress with hooks
  • Admin UI - Plugins can add their own settings pages
my_plugin/src/lib.rs 
use rustpress_plugin::{Plugin, hook};

pub struct MyPlugin;

impl Plugin for MyPlugin {
    fn name(&self) -> &str {
        "My Plugin"
    }

    fn on_init(&self) {
        hook!("after_content", |ctx| {
            // Add custom logic here
        });
    }
}
