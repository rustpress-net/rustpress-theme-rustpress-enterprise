Alternative Installation

Install using your preferred package manager or from source

Cargo (Rust)

cargo install rustpress

Homebrew (macOS)

brew install rustpress/tap/rustpress

Docker

docker pull rustpress/rustpress:latest

From Source

git clone https://github.com/rustpress/rustpress && cd rustpress && cargo build --release

System Requirements

Minimum

  • 1 CPU core
  • 512 MB RAM
  • 100 MB disk space
  • PostgreSQL 13+ or SQLite

Recommended

  • 2+ CPU cores
  • 2 GB RAM
  • 1 GB disk space

Production

  • 4+ CPU cores
  • 4+ GB RAM
  • SSD storage
  • PostgreSQL 15+ cluster
  • Redis 7+ cluster
