What is RustPress?

RustPress is a complete content platform that combines the power of Rust with modern AI capabilities. It's designed to be fast, secure, and infinitely extensible.

Lightning Fast

Built with Rust for blazing-fast performance. Pages load in milliseconds, not seconds. Handle thousands of concurrent users without breaking a sweat.

Secure by Default

Rust's memory safety guarantees eliminate entire classes of vulnerabilities. Built-in protection against SQL injection, XSS, and CSRF attacks.

AI-Powered

The first CMS with built-in AI tools. Generate content, optimize SEO, create images, and enhance your workflow with intelligent assistance.

Developer First

Built by developers, for developers. Clean APIs, intuitive design, and a powerful CLI. Extend everything with custom code.

Extensible Ecosystem

Themes, plugins, functions, and apps. Build custom extensions or choose from hundreds in the marketplace. Make RustPress truly yours.

Free for Everyone

The core platform is completely free. Pro, Business, and Enterprise versions available for advanced features, priority support, and team collaboration.

#[rustpress_plugin]
pub struct MyPlugin;

impl Plugin for MyPlugin {
    fn name(&self) -> &str { "my-plugin" }

    fn on_init(&self, app: &mut App) {
        app.add_route("/api/custom", my_handler);
        app.add_hook("post_save", on_post_save);
    }
}
use rustpress::functions::*;

#[function(trigger = "http", path = "/api/hello")]
async fn hello(req: Request) -> Response {
    let name = req.query("name").unwrap_or("World");
    Response::json(json!({
        "message": format!("Hello, {}!", name)
    }))
}
